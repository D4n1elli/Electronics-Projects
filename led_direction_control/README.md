# LED direction control ⬅️💡➡️
Small Arduino project made by me 😺

## Description
This project goal is to use 2 buttons to control the direction the LED will move, going left or right. For an example, if the user press the left button, the LED on will move to left until the button is released.

## Components
- Arduino UNO
- Jumper wires
- 6 LEDs (you can add as many as you wish)
- Resistors (one for each LED)
- 2 buttons
- Active or passive buzzer

## How to make it yourself
- Place the LEDs on the breadboard, connecting them to GND and to a digital pin through a resistor.
- Connect the 2 buttons to 2 digital pins and to GND.
- Connect the buzzer to a digital pin and to GND.
- Modify the variables in the start of the code according to the pins you placed the components
- Connect your Arduino to your PC, select the model and the port in the Arduino IDE and run the code
