# First solder
This is my first time using a soldering iron and building a circuit in a prototype board

## What I built
A very simple circuit with 2 LEDs, 2 resistors and using an Arduino as power source (no coding required)

## Components I used
- Prototype board
- 2 LEDs
- 2 resistors
- 2 jumper wires
- Paperclip (used as wire)
- Soldering iron and solder

## How it works
Each LED is connected to a resistor, and each resistor is connected to a wire (paperclip)
Both LEDs negative side are connected together and put on gnd

## What I learned
- How to solder a component in a board
- How to use a prototype board

## Mistakes I made
- Adding too many solder
- Resting the soldering iron on the board and slightly burning it
