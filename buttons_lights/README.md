# Buttons Lights 🔴🟡🟢🔵

## Description
This project includes 4 buttons and 4 leds, making a specific LED light up when its button is pressed.

## How it works
The user must press the button corresponding to the LED they want to turn on, multiple LEDs can be lighted up at the same time

## Components
- Arduino UNO
- Breadboard
- Jumper wires
- 4 LEDs (you can use 4 different colors for better visual result)
- 4 resistors
- 4 buttons (if you are using different colors, better use the same button colors to indicate which LED will be lighted up)

## Circuit
Each LED is connected to GND and to a digital pin through a resistor and a jumper wire, the buttons are connected to GND and to digital pins. Place the leds and the buttons next to each other for better visual result

## Possible improvements
- Adding more LEDs and buttons
- Using a buzzer to make a sound each time a LED is pressed
